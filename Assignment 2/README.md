# CAR GAME

Install following

```
npm install
npm install three
npm install webpack
sudo npm -g install servez
```

run following when you change something in any file

```
npx webpack --config webpack.config.js

```
To start the server

```
servez .
```
To play the Game
```
1. Instructions to start -UP ARROW KEY
2. Instructions to play - 
    i) Arrow Keys for Movement
    ii) T to toggle view
```
